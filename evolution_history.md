# evolution_history.md

## 2025-12-23: Web 特化型プロフェッショナル・ダッシュボードの再定義

- [P] LINE 連携をスコープ外とし、Web での「マーケットスピード II」超えの操作性を目標に設定。
- [D] 4 ペイン構成の確定。Zustand による状態管理と JetBrains Mono による数値表示ロジックの設計。
- [C] 4 ペインは情報密度が高いため、14 インチ以下のノート PC での視認性を Shadcn UI の Density（密度）調整で解決予定。
- [A] 銘柄詳細取得 API を「高速・非同期（httpx）」で再実装する。

## 2025-12-23: TradeInfo v3 の完成（実データ・マルチカテゴリ・VWAP）

- [P] 外部スクレイピングによる日足データの完全同期と、マルチカテゴリによる銘柄整理機能を実装。
- [D] 4 カラム構成（ジャンル、リスト、指標、チャート＆ニュース）によりプロ仕様の情報密度を実現。
- [C] VWAP の描画形式（棒グラフから折れ線へ）をユーザーのフィードバックに基づき最適化。

## 2025-12-24: TradeInfo v3.1 視認性向上とレビュー・システムの導入

- [P] ユーザーフィードバックに基づく UI の大幅な整理。ウオッチリストをスッキリさせ、指標を中央パネルへ集約。
- [D] 開発・検証を効率化する「インタラクティブ・レビュー・ダッシュボード」を導入。アプリ内で直接 FB のやり取りを実現。
- [C] マーケット・ティッカーの配色をライトテーマに変更し、1 分ごとの更新を停止。ユーザーの集中力を削がない設計へ。
- [A] Kabutan スクレイピング・ロジックを刷新。表形式の差異を吸収する堅牢な抽出方式により、配当利回りや各種日程の取得精度を極限まで向上。
